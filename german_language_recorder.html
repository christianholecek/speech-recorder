Teilnehmerinformation & Einwilligung

Vielen Dank für Ihre Teilnahme!
Bitte lesen Sie die folgenden Informationen und Anweisungen sorgfältig durch, bevor Sie beginnen.

Was Sie tun werden: Sie lesen kurze Sätze laut vor und nehmen diese auf. Die Sitzung dauert etwa 20 Minuten.

Sie können die beiden Aufgaben in beliebiger Reihenfolge durchführen (zuerst Aufgabe A oder zuerst Aufgabe B).

Vor dem Start (wichtig):

  • Bitte verwenden Sie einen Laptop oder Desktop-Computer (kein Smartphone oder Tablet).
  • Idealerweise sollten Sie sich alleine in einem ruhigen Raum befinden (Fenster schließen, Benachrichtigungen stumm schalten).
  • Sitzen Sie nah am Mikrofon (ca. 30–50 cm) und sprechen Sie deutlich und mit normaler Lautstärke.
  • Für jede Aufnahme: Drücken Sie Aufnahme starten, warten Sie eine Sekunde, sprechen Sie dann. Warten Sie nach dem Sprechen eine Sekunde und drücken Sie dann Stopp.

Mikrofonberechtigung: Ihr Browser wird Sie um Erlaubnis bitten, auf Ihr Mikrofon zuzugreifen. Bitte klicken Sie auf Zulassen. Auf der Mikrofon-Einrichtungsseite können Sie eine Testaufnahme machen, um Ihr Mikrofon zu überprüfen. Testaufnahmen werden nicht gespeichert.

Aufnahmen speichern: Klicken Sie nach jedem Satz auf Akzeptieren & speichern. Ihre Aufnahme wird hochgeladen (das kann einige Sekunden dauern). Sobald der Upload abgeschlossen ist, können Sie zum nächsten Satz weitergehen.

Daten: Audioaufnahmen und Antworten werden sicher gespeichert und für Forschungszwecke verwendet.

Freiwilligkeit: Die Teilnahme ist freiwillig. Sie können jederzeit vor der Einreichung zurücktreten.
